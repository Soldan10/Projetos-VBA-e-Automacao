# Projetos-VBA-e-Automacao
Repositório destinado ao armazenamento de projetos desenvolvidos durante o curso de Análise e Desenvolvimento de Sistemas na FATEC Araraquara. O foco principal é a aplicação de lógica de programação para resolver problemas reais e criar interfaces interativas.

Jogo da Memória Interativo
Descrição 
Um jogo de memória funcional desenvolvido inteiramente dentro do Microsoft Excel.

PT-BR: Utiliza lógica de matrizes (arrays) para o sorteio aleatório das cartas e manipulação de eventos para controlar a interface do usuário.

Sistema de Gestão 
Descrição 
Ferramenta de automação voltada para a gestão de dados e processos.

PT-BR: Implementação de UserForms para entrada de dados, validação de campos e organização automática de planilhas.

Tecnologias e Habilidades 
VBA (Visual Basic for Applications).

Lógica de Programação / Programming Logic.

Microsoft Excel Avançado.

Documentação de Software.
